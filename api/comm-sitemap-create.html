Generate Sitemap XML API

Generate a valid sitemap.xml from an array of URLs. Each URL entry can specify last-modified date, change frequency (daily/weekly/monthly), and priority (0.0–1.0). Automatically splits into sitemap index for large sets. Available as a simple REST API — POST your input, get structured JSON back. 1 credit per call, no rate limiting, works in any language with HTTP support.

Try it live →

How it works

POST a JSON object with your input data to /v1/comm-sitemap-create. The API processes your request server-side using real computation (not mocked data) and returns a structured JSON response with your results and metadata including credits used and processing time.

Use cases

  • Use Generate Sitemap XML in automated pipelines without installing a library
  • Call Generate Sitemap XML from AI agent workflows via MCP or REST
  • Integrate Generate Sitemap XML into serverless functions with zero dependencies
  • Build Generate Sitemap XML functionality into any language that supports HTTP
