How it works
Three steps from raw memory to production-ready skill.
Step 1

Connect your agent

Your agent uses Slopshop memory APIs to store observations, outcomes, and context in its :dreams namespace during every session.

Step 2

Dream Engine extracts patterns

The Dream Engine runs a forecast or evolve strategy — REM-style consolidation that identifies "if X → do Y" patterns above a confidence threshold.

Step 3

Skills appear here

Each extracted skill shows up in this library — ready to copy-paste into your agent's system prompt, deploy as an API call wrapper, or dismiss if irrelevant.
