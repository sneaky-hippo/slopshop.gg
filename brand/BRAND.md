# Slopshop Brand Assets

## Colors
- Primary Red: `#ff3333` (RGB 255, 51, 51)
- Background: `#050505`
- Surface: `#0f0f0f`
- Text: `#f5f5f5`
- Text Secondary: `#999`
- Text Muted: `#666`

## Fonts
- Code/UI: JetBrains Mono (weight 400-800)
- Body: Inter (weight 400-900)

## Logo
- `brand/logo-mark.svg` - Icon only (red square with {·} brackets)
- `brand/logo-full.svg` - Icon + SLOPSHOP.GG wordmark
- `brand/logo-dark-bg.svg` - Full logo on dark background (for social/presentations)

## Logo Concept
The logo is curly braces `{ }` with a dot in the center - representing:
- `{` `}` = API/JSON/code
- `·` = the single point of access (1 API key → everything)
- Red background = Slopshop brand color

## Usage
- Always use on dark backgrounds
- Minimum size: 24px for mark, 120px for full logo
- Don't stretch, rotate, or change colors
