Privacy Policy

Last updated: March 30, 2026

Zero Training Guarantee

We never use your inputs, outputs, memory data, agent runs, or API traffic to train, fine-tune, or improve any language models. Your data stays 100% yours.

What we collect

When you create an account, we collect your email address and generate an API key. When you make API calls, we log the endpoint called, timestamp, credit cost, and latency for billing and debugging. We do not log request or response payloads for compute endpoints.

Agent memory data

Data stored via the memory APIs (memory-set, memory-get, memory-search, memory-list, memory-delete, memory-stats, memory-namespace-list, counter-get) is stored in our database and associated with your API key. You can delete all your memory data at any time via the memory-delete API or by contacting support. For full data sovereignty, self-host Slopshop and all data stays on your infrastructure.

Self-hosting

925 of the 1,255 handlers are pure-compute with zero external dependencies. When self-hosted, no data is sent to Slopshop servers. You control everything.

Third-party services

We use Stripe for payment processing. Stripe's privacy policy applies to payment data. We do not store credit card numbers.

Cookies

We use minimal session cookies for authentication on the dashboard. No tracking cookies. No analytics scripts on the API.

Data retention

API call logs are retained for 90 days. Memory data is retained until you delete it or close your account. Account data is retained until you request deletion.

Data deletion

Email dev@slopshop.gg to request full account and data deletion. We process requests within 48 hours.

Changes

We may update this policy. Changes will be posted here with an updated date.

Contact

Questions about privacy: dev@slopshop.gg
