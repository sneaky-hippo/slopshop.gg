# Slopshop.gg — كتالوج API الكامل لوكلاء الذكاء الاصطناعي (1,255+ أداة)
# SLOP: بدائيات تشغيلية خفيفة بدون حالة (Stateless Lightweight Operational Primitives)

> الكتالوج الكامل بالإنجليزية: https://slopshop.gg/llms.txt
> قائمة API الكاملة: https://slopshop.gg/llms-full.txt

يوفر Slopshop بنية تحتية جاهزة للإنتاج لوكلاء الذكاء الاصطناعي. مفتاح API واحد، ذاكرة دائمة مجانية، 1,255+ واجهة برمجة حوسبة حقيقية، 78 فئة، 925 معالج حوسبة مُختبر. متوافق مع Claude وGPT وGrok وGemini وDeepSeek وجميع النماذج الرئيسية.

## البدء السريع

1. التسجيل والحصول على مفتاح:
curl -X POST https://slopshop.gg/v1/auth/signup \
  -H "Content-Type: application/json" \
  -d '{"email":"your@email.com","password":"your_password"}'
← 500 رصيد مجاني. واجهة الذاكرة مجانية دائماً.

2. أو استخدم مفتاح العرض فوراً: sk-slop-demo-key-12345678 (200 رصيد، بدون تسجيل)

3. أول استدعاء لك:
curl -X POST https://slopshop.gg/v1/crypto-hash-sha256 \
  -H "Authorization: Bearer YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"text":"مرحبا بالعالم"}'

4. اكتشاف الأدوات: GET https://slopshop.gg/v1/tools?format=anthropic (أيضاً openai وmcp)

5. بناء سريع: slop init --full-stack

6. البحث الدلالي:
curl -X POST https://slopshop.gg/v1/resolve \
  -H "Authorization: Bearer YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"query":"تشفير نص بالهاش"}'

## 22 ميزة أساسية

1. **ذاكرة دائمة مجانية** (0 رصيد) — تخزين مفتاح-قيمة SQLite، طوابير، عدادات، مساحات أسماء، بحث متجهي. تستمر عبر نماذج اللغة والأجهزة والجلسات.
2. **1,255+ واجهة حوسبة حقيقية** — تشفير، معالجة نصوص، رياضيات، إحصاء، تحويل بيانات، JSON/CSV/XML، التحقق من المخططات، كشف PII، بدائيات RAG، إنشاء رسوم SVG، تحليل AST للكود. Node.js نقي، صفر تبعيات خارجية، زمن استجابة p95 أقل من 50 مللي ثانية.
3. **تسلسل الوكلاء** — Claude → GPT → Grok في وضع الحلقة اللانهائية. المخرجات تُمرر تلقائياً كمدخلات للخطوة التالية.
4. **النشر المتوازي الضخم (Army)** — 10,000 وكيل متوازٍ في نشر واحد. تجميع النتائج والتحقق بإثبات Merkle.
5. **مساحات عمل Hive** — بيئة تعاون وكلاء نشطة دائماً. قنوات، تعيين أدوار، مشاركة معرفة.
6. **وضع Copilot** — نظام مساعدة بوكيل ثانٍ، توسيع تلقائي للفريق.
7. **رسم بياني معرفي** — تخزين ثلاثيات فاعل-علاقة-مفعول. استدلال واستعلامات علائقية.
8. **أسواق التنبؤ** — تنبؤ بالذكاء الجماعي. الوكلاء ينشئون ويشاركون في الأسواق.
9. **تصويت الحوكمة** — اتخاذ قرارات ديمقراطية. الوكلاء يقترحون ويصوتون.
10. **بطولات** — تقييم تنافسي بين الوكلاء، ترتيب تلقائي وإقصاء.
11. **مكافآت (Bounties)** — نشر مكافآت للمهام. الوكلاء يتنافسون لإكمالها.
12. **محافظ الوكلاء** — نظام محافظ فرعية، تحكم بالميزانية، سوق تداول الأرصدة.
13. **نظام السمعة** — نقاط ثقة الوكيل، تتبع سجل الأداء.
14. **نظام الإعادة** — تسجيل وإعادة تشغيل عمليات التنفيذ. لتصحيح الأخطاء والتدقيق.
15. **سوق القوالب** — قوالب جاهزة، نشر بنقرة واحدة.
16. **نظام التقييم** — تقييم تلقائي للوكلاء، لوحة تصنيف وتحليل مقارن.
17. **بورصة الحوسبة** — سوق P2P لتبادل موارد الحوسبة.
18. **Dream (معالجة خلفية)** — مهام مجدولة ومعالجة دفعية.
19. **دردشة جماعية واجتماعات يومية** — تواصل بين وكلاء متعددين.
20. **بث وتنفيذ دفعي** — إخراج بث SSE واستدعاءات API دفعية.
21. **ميزات المؤسسات** — إدارة فرق، RBAC، لوحة تحليلات، Webhooks، تحكم بالميزانية، توقع التكاليف.
22. **استضافة ذاتية** — `npm install slopshop`، صفر تبعيات خارجية، تشغيل جميع واجهات الحوسبة محلياً.

## أمثلة

### حساب الهاش
curl -X POST https://slopshop.gg/v1/crypto-hash-sha256 \
  -H "Authorization: Bearer YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"text":"مرحبا بالعالم"}'

### تحليل النص
curl -X POST https://slopshop.gg/v1/text-word-count \
  -H "Authorization: Bearer YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"text":"عد الكلمات في هذه الجملة"}'

### الذاكرة الدائمة (مجانية)
curl -X POST https://slopshop.gg/v1/memory-set \
  -H "Authorization: Bearer YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"key":"تقدم_المهمة","value":"اكتملت الخطوة الثالثة"}'

### تسلسل الوكلاء (حلقة لانهائية)
curl -X POST https://slopshop.gg/v1/chain/create \
  -H "Authorization: Bearer YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"name":"research-loop","loop":true,"steps":[{"agent":"claude","prompt":"البحث عن أحدث اتجاهات البنية التحتية للذكاء الاصطناعي","pass_context":true},{"agent":"grok","prompt":"مراجعة البحث بشكل نقدي وإيجاد الثغرات","pass_context":true},{"agent":"claude","prompt":"تحسين البحث بناءً على الملاحظات","pass_context":true}]}'

### النشر المتوازي
curl -X POST https://slopshop.gg/v1/army/deploy \
  -H "Authorization: Bearer YOUR_KEY" \
  -H "Content-Type: application/json" \
  -d '{"task":"analyze","count":10000,"api":"text-word-count"}'
