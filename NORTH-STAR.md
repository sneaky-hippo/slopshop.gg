## 3. ARCHITECTURE PRINCIPLES

1. **Pipe, Don't Trap.** Every output is a valid input to something else. Dead ends are bugs.
2. **Model-Agnostic by Default.** No API requires a specific LLM. We outlive every model generation.
3. **Memory Is Infrastructure.** Free, foundational, available to every API and orchestration flow.
4. **Self-Hostable or It Doesn't Ship.** Cloud is convenience. Self-host is the product.
5. **CLI Is the Control Plane.** Breadth lives in APIs; CLI stays ruthlessly small and elegant.
