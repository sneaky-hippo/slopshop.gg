## What it does
- 1272 real API endpoints across 78 categories (925 compute handlers)
- Native MCP server (slop mcp serve) — works with Claude Desktop, Cursor, Goose, OpenCode, Cline, Aider
- Multi-LLM orchestration (Claude, GPT, Grok, DeepSeek, Ollama)
- Free persistent memory for AI agents (8 APIs, 0 credits forever)
- Memory 2.0: GraphRAG graph queries, auto-summarize namespaces, snapshot and pin
- Memory Evolution: autonomous consolidate, enrich, decay, summarize strategies (POST /v1/memory/evolve/start)
- Chain Branching: conditional step execution with if/else_step (POST /v1/chain/run/branching)
- Eval Benchmark: automated testing with 67 test vectors (POST /v1/eval/benchmark)
- Agent organizations (launch 30-agent teams in one call)
- Army Mode: 10,000 parallel agents with Merkle-verified results
- Hive Workspaces: persistent multi-agent collaboration
- Credit System: agents earn credits by sharing idle compute
- Browser/Desktop primitives: Swarm Visualizer SSE, MCP Desktop bootstrap
- Safety layer: sandbox execution with vm.createContext isolation, circuit breakers, chaos testing
- Marketplace: template browsing, bounty board, Plugin Forge for custom tools
- Agent templates: research swarm, content machine, security audit, data pipeline
- Grok autonomous orchestration: optimize, critique, overlord endpoints
- Official integrations: Goose Recipes, Aider commands, OpenCode plugin, Cline Skills, LangChain adapters
- Self-hostable, zero native dependencies

## Key endpoints
- POST /v1/llm-think — Ask any LLM to reason (provider=anthropic|openai|grok|deepseek)
- POST /v1/llm-council — Get all LLMs to answer the same question
- POST /v1/compare — Side-by-side multi-LLM comparison with ranking
- POST /v1/agent/run — Autonomous agent that picks tools and executes
- POST /v1/org/launch — Launch multi-agent organizations
- POST /v1/workflows/run — Declarative multi-step conditional chains
- POST /v1/context/session — Get execution context for LLMs
- GET /v1/introspect — Discover any API schema
- POST /v1/route — Smart task-to-API routing
- POST /v1/memory-set — Free persistent key-value memory
- POST /v1/memory/graph-query — GraphRAG graph queries across memory
- POST /v1/grok/optimize — Grok-driven optimization
- POST /v1/grok/critique — Grok-driven critique
- POST /v1/grok/overlord — Autonomous multi-LLM orchestration
- POST /v1/marketplace/browse — Browse agent templates and plugins
- POST /v1/sandbox/exec — Sandboxed code execution with vm.createContext isolation

## MCP Integration
Add to Claude Code / Cursor / any MCP client:
{
  "mcpServers": {
    "slopshop": {
      "command": "npx",
      "args": ["slopshop", "mcp", "serve"],
      "env": { "SLOPSHOP_KEY": "sk-slop-your-key-here" }
    }
  }
}

## Install (CLI)
npm install -g slopshop
slop signup
slop call crypto-uuid
slop mcp serve          # Start MCP server
slop init --full-stack  # Scaffold project
