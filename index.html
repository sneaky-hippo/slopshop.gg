Dream Engine

Nine strategies. One overnight run.

Memory doesn't improve by just storing more. Dream Engine runs nine sequential cognitive strategies — synthesize, pattern_extract, insight_generate, compress, associate, validate, evolve, forecast, reflect — producing cross-domain insights your agent didn't have before.

Schedule it nightly, trigger it programmatically, or let agents self-schedule. Each run returns a full report with Intelligence Score, top insights, and synthesized knowledge clusters.

GET /v1/memory/dream/report/:id
