Developers

Built on a real API.

REM Labs is the consumer layer. Underneath is slopshop.gg — a full REST API, CLI, MCP server, and 1,303 tool endpoints. Everything you see here is accessible via code.

# Store a memory curl -X POST https://api.slopshop.gg/v1/memory/store \ -H "Authorization: Bearer $SLOP_KEY" \ -d '{"content":"User prefers concise responses."}' # → { "id": "mem_9a3z1", "stored": true } # Start a dream cycle curl -X POST https://api.slopshop.gg/v1/memory/dream/start \ -H "Authorization: Bearer $SLOP_KEY"
# Install npm install -g @slopshop/cli # Store from stdin echo "prefers dark mode" | slop memory store # Run a dream cycle slop dream run --all-stages
